ledr

Introduction

Ledr is a plain text accounting tool, written in Rust. It is designed for efficient personal and professional accounting for the discerning user of command line tools, for individuals and organizations alike.

Documentation

Ledr's documentation is in progress. At the moment is is not ready for publication, and ledr's structure and commands are not yet stable enough to fully document. Documentation and stability is expected to be achieved by Q2 2025. When ready, this website will become the documentation hub for the tool.

Development

Ledr is developed by Adam House. As of November 2024, it is pre-alpha software. Using it now means participating in its development. Check out the source code on GitHub.
